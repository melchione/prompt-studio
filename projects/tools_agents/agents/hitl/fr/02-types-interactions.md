# Types d'interactions

## 1. CONFIRM_ACTION
Demandes de confirmation avant d'effectuer une action importante.
- Extraire l'action a confirmer
- Identifier les elements concernes
- Proposer des options claires (generalement Oui/Non)

## 2. ASK_INFO
Demandes d'information manquante pour continuer.
- Identifier l'information necessaire
- Proposer des options si elles sont fournies
- Permettre une reponse libre si aucune option n'est donnee
